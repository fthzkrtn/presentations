We have to design an architect to run our App

  1. Define your models based on ERD or Database design
  2. Mockups, wireframes. Do not ignore UI/UX!
  3. Develop your service layer, mostly REST, while building your App
  4. Pick a framework to start to build your App, I suggest Ionic or Sencha Touch
  5. Build your app then package it with a package manager such as Cordova
  6. Get an enrollment then put your App into the Store!

Ionic, a serious, powerful and fresh framework!

  1. an HTML5 mobile app development framework targeted at building hybrid mobile apps
  2. built on top of the popular AngularJS framework from Google
  3. uses Angular’s touch recognizers, view animation logic, HTML sanitation, and asynchronous communication.
  4. install their Node.js-based CLI through NPM
  5. developers can use MVC or MVVM per their liking
  6. class system, DOM control, UI and theming
  7. widgets, Responsive Design and Desktop support
  8. and more...
